# EventHorizon

A classy Event Management System built with Flask.

## Features
- **Create Events**: Admin interface to add new events.
- **View Events**: Styled table display with "Classy" UI.
- **Update/Delete**: Full CRUD functionality with icon controls.
- **Register**: User registration for events.

## Tech Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML/CSS (Custom "Classy" Design System)

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install flask
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Visit `http://127.0.0.1:5000` in your browser.
